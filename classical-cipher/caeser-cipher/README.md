# Caesar-Cipher
## Requirements
* Python 3 version 3.5
* Apple LLVM version 10.0.0

## Usage
### Python 3
On your console:

```sh
$ python3 enc.py
```
Input key and plain text.

```sh
$ python3 dec.py
```
Input key and encrypted text.

### C++
On your console:
```sh
$ clang++ enc.cpp
$ ./a.out
```
Input key and plain text.

```sh
$ clang++ dec.cpp
$ ./a.out
```
Input key and encrypted text.
